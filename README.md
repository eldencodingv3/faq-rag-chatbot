# FAQ RAG Chatbot

A Node.js RAG (Retrieval-Augmented Generation) chatbot for customer support. It uses LanceDB for vector storage and OpenAI for embeddings and chat completions to answer questions based on your FAQ dataset.

## Prerequisites

- Node.js 18+
- OpenAI API key

## Quick Start

```bash
git clone https://github.com/eldencodingv3/faq-rag-chatbot.git
cd faq-rag-chatbot
npm install
```

Set your OpenAI API key:

```bash
export OPENAI_API_KEY=your-key-here
```

Start the server:

```bash
npm start
```

The chatbot will be available at `http://localhost:3000`.

## Environment Variables

| Variable | Required | Default | Description |
|---|---|---|---|
| `OPENAI_API_KEY` | Yes | — | Your OpenAI API key |
| `PORT` | No | `3000` | Port the server listens on |

## Updating the FAQ Dataset

Edit `data/faqs.json` with your questions and answers, then restart the server. The embeddings will be regenerated on startup.

## Tech Stack

- **Express.js** — HTTP server and API routing
- **LanceDB** — Local vector database for FAQ embeddings
- **OpenAI GPT-3.5-turbo** — Chat completions and text embeddings

## API Endpoints

### `GET /`

Serves the chatbot web interface.

### `POST /api/chat`

Send a chat message and receive an AI-generated answer based on relevant FAQs.

**Request body:**

```json
{ "message": "How do I reset my password?" }
```

**Response:**

```json
{ "reply": "To reset your password, go to..." }
```

### `GET /api/health`

Health check endpoint. Returns `{ "status": "ok" }`.

## Deployment

The app auto-deploys to Railway on merge to `main`. The production URL is generated by Railway.
